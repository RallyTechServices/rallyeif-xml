# RallyEIF::XML

Filesystem/XML Spoke for EIF Work item connectors

## Installation

Add this line to your application's Gemfile:

    gem 'rallyeif-xml'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rallyeif-xml

## Usage

    $ rally2_xml_connector.rb <config XML> <interval in minutes>
